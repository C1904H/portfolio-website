myFlix Movie API

A server-side component of a "movies" web application. Users can access information about different movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

This RESTful API is a Node.js and Express application.

Technologies used: Express.js, MongoDB, Mongoose, Passport.js, Swagger, CORS, Bcrypt.

myFlix React App

Client-side for myFlix movie app using the existing server-side code (REST API and database). Users can sign up/ login, browse movies (inc description, image, genre and director), select favorite movies or search for movie by title. Users can navigate to their personal profile where they can update their user information, view favorite movies or delete account.

Technologies used: A full-stack JavaScript developement: MERN (MongoDB, Express, React and Node.js) stack.

Meet App

A serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. This application uses the Google Calendar API to fetch upcoming events and OAuth2 authentication flow.

Users can explore events by searching for a city and choose how many events to display.

Technologies used: React, Google Calender API, AWS Lambda, Serverless, Jest, Jest-Cucumber, Puppeteer, Recharts.
